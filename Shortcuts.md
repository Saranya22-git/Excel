### **Keyboard Shortcuts**

| Shortcut      | Action                              |
| ------------- | ----------------------------------- |
| Arrow Keys    | Move one cell                       |
| Tab           | Move right                          |
| Shift + Tab   | Move left                           |
| Enter         | Move down                           |
| Shift + Enter | Move up                             |
| Home          | Move to Column A of the current row |
| Ctrl + Home   | Move to A1                          |
| Ctrl + End    | Move to the last used cell          |
| Ctrl + N      | A new workbook opens instantly      |
